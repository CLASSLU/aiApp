# 部署指南

## 环境要求
```yaml
# docker-compose.yml 中的服务配置
services:
  backend:
    build: ./src
    environment:
      - SILICONFLOW_API_KEY=${API_KEY}
    ports:
      - "5000:5000"

  frontend:
    build: ./frontend
    ports:
      - "80:80"
```

## 持续集成
```bash
# 示例部署脚本
export API_KEY=your_production_key
docker-compose build
docker-compose up -d
```

## 监控配置
- Prometheus 指标端点：`/metrics`
- 健康检查端点：`/health`
